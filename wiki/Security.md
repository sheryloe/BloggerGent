# Security

## 기본 보안 원칙

- 민감정보는 설정 저장소 암호화 사용
- 운영 계정 접근 제어 유지
- 발행 작업은 추적 가능하게 기록
- 실패 작업은 재시도/복구 로그를 남김

## 필수 점검 항목

1. `SETTINGS_ENCRYPTION_SECRET` 강도 점검
2. API/대시보드 접근 경로 제한 점검
3. OAuth 토큰 저장/갱신 상태 점검
4. 운영 로그(`TEST/logs`) 보관 정책 점검

## 운영 메모

보안은 코드뿐 아니라 운영 절차까지 포함합니다.
변경 작업마다 테스트 로그와 변경 이력을 남겨 재현 가능성을 확보합니다.
