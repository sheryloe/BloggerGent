# SEO-Metadata

Donggr AutoBloggent는 SEO 메타를 "입력 완료"가 아니라 "공개 후 검증 완료" 기준으로 운영합니다.

## 핵심 관점

- `description`
- `og:description`
- `twitter:description`

위 3개가 실제 공개 페이지에서 기대값과 일치하는지 확인합니다.

## 점검 순서

1. 글 공개 상태 확인
2. 메타 설명/본문 의도 일치 확인
3. 필요 시 search description 동기화
4. 재검증 후 상태 업데이트

## 운영 팁

- 점수/상태는 의사결정 신호로 사용
- 본문 품질과 메타 품질을 분리해서 관리
- 메타 이상은 발행 후 빠르게 재검증
