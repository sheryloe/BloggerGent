# Workflow

Donggr AutoBloggent 운영은 대시보드 기준 4개 묶음으로 관리합니다.

## 1) 생성

- 주제 발굴
- 초안 생성
- 기본 메타데이터 생성

## 2) 자산

- 이미지/썸네일 생성
- 본문 내 자산 정리
- 누락 자산 보강

## 3) 업로드

- 게시 큐 확인
- 예약/발행 실행
- 실패 항목 재큐잉

## 4) 운영

- 런타임 상태 점검
- 알림/실패 런 대응
- Analytics/Google 피드백 반영

## 운영 원칙

- 생성과 공개 발행은 분리
- 오류 복구 가능성을 우선
- 대시보드에서 즉시 이동 가능한 동선 유지
