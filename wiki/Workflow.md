# Workflow

## 4개 운영 묶음
1. 생성: 주제/초안 생성
2. 자산: 이미지/썸네일/본문 보강
3. 업로드: 예약/게시 큐 처리
4. 운영: 알림/실패/재시도 제어

## 화면 매핑
- Mission Control: 전체 흐름 점검
- Planner: 타입 -> 채널 기준 월간/일간 계획
- Content Lab: 타입별 콘텐츠 운영
- Analytics: 채널별 성과 확인
- SEO / Indexing: 블로그 탭 기준 색인 관리

## 운영 권장 순서
1. Integrations 점검 (`/settings`)
2. Planner 계획 확인 (`/planner`)
3. Content Lab 실행 (`/content-ops`)
4. Ops Monitor 점검 (`/ops-health`)
5. Analytics/SEO 피드백 반영
