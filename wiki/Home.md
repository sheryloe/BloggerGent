# Bloggent Wiki

Bloggent is a Blogger publishing workspace for operators who want AI assistance, manual publishing control, and real metadata verification on the public page.

## Quick Answer

Use this wiki if you want the shortest path to understanding how Bloggent works today:

1. boot the local stack
2. connect Google
3. import Blogger blogs
4. generate content
5. review in `/articles`
6. publish manually
7. verify metadata on the live post
