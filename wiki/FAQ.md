# FAQ

## 발행은 완전 자동인가요?

기본 원칙은 **자동 생성 + 운영자 제어 발행**입니다.
운영 안전성을 위해 공개 전 최종 확인 단계를 유지합니다.

## 대시보드는 어디가 메인인가요?

`http://127.0.0.1:7001/`로 접속하면 `/dashboard`로 이동하며, 이 화면이 운영 메인입니다.

## 4개 묶음은 무엇인가요?

- 생성
- 자산
- 업로드
- 운영

대시보드에서 이 순서로 실행 흐름을 관리합니다.

## GitHub Pages를 어떻게 쓰나요?

GitHub Pages는 정적 공개 경로/문서 보조 용도입니다.
실제 운영 화면은 `web` 서비스(`/dashboard`)를 기준으로 사용합니다.

## 브랜드명은 무엇인가요?

웹/기본 설정/API 기본 표시는 모두 `Donggr AutoBloggent`를 기준으로 맞춥니다.
