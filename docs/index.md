# Bloggent 문서

Bloggent는 여러 Blogger 블로그를 한 워크스페이스에서 운영하면서, AI 주제 발굴, GEO + SEO 글 생성, 수동 발행, 공개 메타 검증까지 연결하는 운영 플랫폼입니다.

## 제품 핵심 정리

1. 블로그별 워크플로와 프롬프트를 분리합니다.
2. 생성과 발행을 분리해 수동 검토를 보호합니다.
3. GEO + SEO 구조로 글 패키지를 만듭니다.
4. 공개 페이지 기준으로 메타 반영 상태를 검증합니다.
