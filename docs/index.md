# Bloggent 문서

Bloggent의 기본 운영 문서는 Cloudflare R2 기반 이미지 전달 전략을 기준으로 정리되어 있습니다.

## 먼저 볼 문서

- [시작하기](getting-started.html)
- [배포](deployment.html)
- [워크플로](workflow.html)
- [SEO 메타데이터](seo-metadata.html)

## 핵심 메모

- 공개 이미지 기본 provider: `cloudflare_r2`
- 운영 권장 도메인: `img.<domain>`
- 원본 저장: Cloudflare R2
- 렌더링 최적화: `/cdn-cgi/image`
