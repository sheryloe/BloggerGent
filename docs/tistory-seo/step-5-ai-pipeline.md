# Step 5. AI 글 생성과 이미지 생성 파이프라인 구현

## 추천 제목
- Blogger SEO 자동화의 핵심: AI 글 생성과 이미지 파이프라인 구현기
- OpenAI와 Gemini로 블로그 글 자동 생성 파이프라인 만들기
- 주제 발굴부터 HTML 조립까지, AI 블로그 파이프라인 구조 공개

## 검색 설명
Gemini 주제 발굴, OpenAI 글 생성, 이미지 프롬프트 생성, 대표 이미지 생성, HTML 조립까지 Blogger 자동화 파이프라인을 단계별로 정리합니다.

## 추천 태그
`OpenAIAPI`, `GeminiAPI`, `AI글쓰기`, `이미지생성`, `자동화파이프라인`

## 글 구조
- 주제 발굴 단계
- 본문 생성 단계
- 이미지 프롬프트 / 이미지 생성 단계
- HTML 조립과 관련 글 삽입
- 블로그별 워크플로가 중요한 이유

## 본문 샘플

이번 프로젝트에서 중요한 포인트는 “AI를 한 번만 부르는 것”이 아니라 “어떤 단계를 어떤 순서로 묶을 것인가”였습니다. 같은 주제라도 여행 블로그와 미스터리 블로그는 글 쓰는 방식이 다르기 때문입니다.

그래서 글 생성 파이프라인을 고정 기능으로 두지 않고, 블로그별로 단계 순서와 프롬프트를 저장하는 구조로 만들었습니다. 이렇게 해야 서비스형으로 확장할 수 있습니다.

## HTML 예시

```html
<h2>파이프라인 단계</h2>
<ul>
  <li><strong>주제 발굴</strong>: Gemini로 키워드 후보 수집</li>
  <li><strong>본문 생성</strong>: OpenAI로 SEO 구조 글 작성</li>
  <li><strong>이미지 프롬프트</strong>: 콜라주용 프롬프트 생성</li>
  <li><strong>이미지 생성</strong>: 대표 이미지 생성</li>
  <li><strong>HTML 조립</strong>: 본문, FAQ, 관련 글 결합</li>
  <li><strong>Blogger 발행</strong>: draft 또는 publish</li>
</ul>
```

## 마무리 문장 예시

글을 자동으로 만드는 것만으로는 운영이 끝나지 않습니다. 실제 서비스처럼 쓰려면 상태 관리와 관리 화면이 중요합니다. 다음 글에서는 대시보드와 설정 UX를 어떻게 정리했는지 소개하겠습니다.
