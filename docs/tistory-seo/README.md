# BloggerGent 티스토리 SEO 시리즈

이 폴더는 `BloggerGent`를 실제로 구현하고 운영하면서 정리한 티스토리용 원고 모음입니다.
목표는 단순 회고가 아니라, 검색 유입이 가능한 기술형 시리즈 문서로 정리하는 것입니다.

구성:

1. `step-1-keyword-and-problem.md`
2. `step-2-architecture.md`
3. `step-3-docker-and-local.md`
4. `step-4-google-oauth-and-blogger.md`
5. `step-5-ai-pipeline.md`
6. `step-6-dashboard-and-ux.md`
7. `step-7-operations-and-seo.md`
8. `step-8-growth-and-retrospective.md`

활용 방법:

- 티스토리 글 본문 초안으로 그대로 사용
- Notion 하위 페이지로 업로드 후 다듬기
- 각 문서의 `추천 제목`, `검색 설명`, `추천 태그`, `HTML 예시`를 바로 재사용

시리즈 방향:

- 검색 의도에 맞는 키워드 배치
- 구현 과정 + 이론 + 실전 운영 경험을 같이 설명
- 코드만 나열하지 않고 왜 그렇게 설계했는지까지 설명
- 티스토리에서 읽기 쉽도록 짧은 문단과 명확한 중간 제목 구성
